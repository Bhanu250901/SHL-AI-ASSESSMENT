# SHL AI Assessment Recommendation API

## Features

- FastAPI-based conversational recommendation API
- SHL assessment retrieval system
- Clarification question support
- Recommendation refinement support
- Assessment comparison support
- JSON API responses
- Deployment on Render

## Evaluation Methods

The solution was evaluated using:

- Retrieval relevance testing
- Recommendation accuracy testing
- Groundedness verification using SHL catalog URLs
- Manual conversational testing
- API response validation
